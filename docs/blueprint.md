# **App Name**: VendingVerse

## Core Features:

- Product Display: Show available products on the touchscreen with prices and images.
- Touchscreen Input: Enable users to select products using the touchscreen interface.
- Payment Processing: Process payments using various methods (e.g., credit card, mobile payment).
- Inventory Management: Keep track of product quantities in real-time in Firestore.
- Sales Tracking: Record sales data to analyze purchasing patterns, and automatically send that to Firestore.
- Remote Monitoring: Allows you to keep an eye on vending machines that may require maintenance through an AI model. AI can automatically recognize the amount of product inside of the vending machine
- Firestore Integration: Integrate the current application with Firestore for inventory management and sales tracking.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) for a modern and reliable feel.
- Background color: Light grey (#F5F5F5), nearly white, for neutrality and legibility.
- Accent color: Blue (#3F51B5) for interactive elements.
- Body and headline font: 'PT Sans' (sans-serif) for clear readability.
- Use clean and recognizable icons for product categories and payment options.
- Prioritize a clear, intuitive layout for the touchscreen interface.
- Use subtle animations for product selection and confirmation.
